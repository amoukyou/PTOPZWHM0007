七、操作步骤

  1. IBKR账户：开通欧洲期权交易权限，交易所选MEFF。
  2. 搜索合约：在IBKR搜索"IBEX 35"，找Mini IBEX期权（乘数€1/点），到期月2027年3月，类型Put，行权价17,062。
  3. 买入16张：限价单，参考价约€1,035/张，总计约€16,553。
  4. 持有期间：IBEX涨超20%（>20,474）时考虑提前换仓，否则不用管。
  5. 到期前1个月滚仓：卖旧Put，买新的12个月ATM Put，周而复始。

八、局限性

  1. 保费是确定支出：每年€16,553，5年不出事白花€82,765。
  2. 中等回调覆盖有限：IBEX跌5-10%时，16张Mini合约只能覆盖部分损失。跌20%+才有力。
  3. IV影响成本：恐慌期Put更贵，尽量在平静期滚仓。
